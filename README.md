# RabbitMQ 中间件

## 概述

RabbitMQ是一个开源的消息代理软件，实现了高级消息队列协议（AMQP）。它提供了可靠的消息传递、灵活的路由、集群支持等特性。

## 版本信息

- **包版本**: 1.1.0-1.0.0
- **支持版本**: 3.12.7
- **供应商**: harmonycloud
- **类型**: mq (消息队列)

## 功能特性

- 支持多种消息传递模式
- 高可用性和集群支持
- 灵活的路由和交换器
- 消息持久化和确认机制
- 管理界面和监控
- 支持多种协议

## 部署模式

### 标准模式 (Standard)
- 单节点部署
- 适用于开发和测试环境
- 基本功能完整

### 高可用模式 (Highly Available)
- 多节点集群部署
- 支持故障转移
- 适用于生产环境

### 主动-主动模式 (Active-Active)
- 跨可用区部署
- 负载均衡和故障转移
- 适用于大规模生产环境

## 配置参数

### 基础配置
- `version`: RabbitMQ版本
- `replicas`: 副本数量
- `storageSize`: 存储大小
- `logStorageSize`: 日志存储大小

### 资源配置
- `resources.requests`: 资源请求
- `resources.limits`: 资源限制

### 网络配置
- `podAntiAffinity`: Pod反亲和性策略
- `nodeAffinity`: 节点亲和性配置
- `tolerations`: 容忍度配置

## 使用说明

1. 选择合适的部署模式
2. 配置必要的参数
3. 部署到Kubernetes集群
4. 配置连接信息
5. 验证部署状态

## 监控和告警

- 支持Prometheus监控
- 内置告警规则
- 可配置的指标收集

## 故障排除

- 检查Pod状态
- 查看日志信息
- 验证网络连接
- 检查资源配置
